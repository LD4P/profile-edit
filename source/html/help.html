Bibframe Profile Help

How do I create a profile?

On the profile list page, click 'new profile' above the table on the right-hand side.

What is a Resource Template?

A Resource Template construct describes one of the various resources (Works, Instances, etc) associated with a given Profile.

How do I add a resource template?

  1. Navigate to the profile editing page by selecting a profile or clicking 'new profile'.
  2. Click the 'Add Resource Template' button.

What is a Property Template?

A Property Template construct describes one of the various properties associated with a particular resource.

How do I add a property template?

  1. Navigate to the profile editing page by selecting a profile or clicking 'new profile'.
  2. Click the desired resource template.
  3. At the bottom of the list of property templates, click 'add property template'.

How do I add a resource to a property template?

  1. Navigate to the profile editing page by selecting a profile or clicking 'new profile'.
  2. Click the desired resource template.
  3. Click on the desired property template.
  4. At the bottom of the property template, click on 'Add Resource'.
  5. Enter your information into the resource template, or choose an existing one.

How do I use an existing resource/property template?

  1. Navigate to the profile editing page by selecting a profile or clicking 'new profile'.
  2. Click the 'Change Resource/Property' or 'Select Resource/Property' button to open a dialog window.
  3. Select your desired vocabulary list from the menu.
  4. Select an item from the list in the selection area, and your resource/property template will populate itself.

How do I add a value constraint?

A value constraint is automatically added to each property template. If you do not enter any data into these fields, no value constraints will be saved with this property template.

How do I add a value data type?

A value data type is automatically added to each property template. If you do not enter any data into these fields, no value data types will be saved with this property template.

How do I add a template reference?

  1. Navigate to the profile editing page by selecting a profile or clicking 'new profile'.
  2. Expand the desired resource template.
  3. Expand the desired property template.
  4. At the bottom you will see the templates and values.
  5. Click 'Add Template'.
  6. As you begin typing in the template input field, you will be offered suggestions for templates. Choose one of these selections, or continue typing to add a new template reference.

How do I import an existing profile?

  1. Navigate to the profile editing page by clicking 'new profile'.
  2. Click the import button at the bottom.
  3. A dialog box will appear with the text 'Drop file to upload' if you are using Chrome, Firefox, or IE 10 or later.
  4. Drag the file from you computer onto the blue cloud.
  5. If you are using IE 8 or 9, a file dialog will appear when you click the import button.
  6. Select the desired file in the dialog.
  7. The editor will fill in the data or return an error.

How do I export a profile?

  1. Navigate to the profile editing page by selecting a profile or clicking 'new profile'.
  2. If creating a new profile, enter the information into the profile.
  3. Click the export button.
  4. Select a destination for the profile and click save.
  5. A notification will show any errors that are present in the profile.
  6. If the profile does not contain any errors, the editor will save it to your desired destination

How do I save a profile?

Click save at the top of the profile page. If you have errors on your profile you will receive a notification stating that the profile could not be saved.

How do I rename a profile?

  1. Navigate to the profile editing page by selecting a profile.
  2. Open the profile tab.
  3. Change the title.
  4. Select overwrite on the notification.

How do I create a copy of a profile with a new name?

  1. Navigate to the profile editing page by selecting a profile.
  2. Open the profile tab.
  3. Change the title.
  4. Select 'No, make a copy' on the notification.

How do I alter a profile?

  1. Navigate to the profile editing page by selecting a profile.
  2. Enter your changes.
  3. Click save.

Why won't my profile save?

If your profile will not save and a notification appears, it means there is an error in your profile. Check to make sure all required fields (denoted by a red !) are filled in and no error messages exist in your profile.
